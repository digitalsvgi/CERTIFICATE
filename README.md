# CertifyPro - Premium Certificate Generator

Generate professional certificates in bulk with ease.

## Features
- **Excel/CSV Integration**: Import data directly from your spreadsheets.
- **Custom Templates**: Use any image as a certificate template.
- **Live Preview**: Drag and drop text to position it perfectly on your template.
- **Bulk Generation**: Generate hundreds of certificates in seconds.
- **ZIP Download**: All certificates are bundled and downloaded as a single ZIP file.

## How to Use
1. **Open `index.html`** in any modern web browser (Chrome, Edge, Firefox).
2. **Upload your Certificate Template** (PNG/JPG).
3. **Upload your Data File** (XLSX/CSV).
4. **Choose the column** that contains the names.
5. **Adjust the text** position by dragging it on the preview.
6. **Customize** font size, family, and color.
7. Click **"Generate & Download ZIP"**.

## Dependencies
This project uses the following libraries via CDN:
- [XLSX](https://github.com/SheetJS/sheetjs) for spreadsheet parsing.
- [JSZip](https://stuk.github.io/jszip/) for bundling images.
- [FileSaver.js](https://github.com/eligrey/FileSaver.js/) for downloads.
- [Lucide](https://lucide.dev/) for icons.

Enjoy your bulk certificate generation!
